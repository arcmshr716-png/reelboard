# ReelBoard

リールの台本を大画面で表示・管理するミニマルなWebアプリ。

## 機能

- **台本の作成・編集** — タイトルをつけて台本を保存
- **ステージモード** — フルスクリーンの大画面で台本を表示
- **リンク管理** — 台本にリンクを添付
- **画像管理** — 画像URLを登録してステージに表示
- **自動保存** — 編集内容はlocalStorageに自動保存
- **文字サイズ調整** — ステージ中に `+` / `-` キーでサイズ変更

## 使い方

1. `index.html` をブラウザで開く
2. 「新しい台本」ボタンでタイトルと台本テキストを入力
3. 右サイドバーにリンク・画像を追加
4. 「▶ ステージ」ボタンで大画面表示
5. `Esc` キーでステージを終了

## 技術

- 純粋なHTML / CSS / JavaScript（依存なし）
- データはブラウザのlocalStorageに保存
- Google Fonts: Inter + Noto Sans JP
